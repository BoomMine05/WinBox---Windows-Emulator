# WinBox - Windows-Emulator
Install any Windows OS on your Android phones, like a real PC. Based in Qemu & better limbo

WinBox - is a PC emulator for install or run any Windows OS on your Android phones.
# How to install #

1. Download a WinBox.apk from releases;
2. Install it;
3. Open an app;
4. Agree a grand permission to media, photos and videos;
5. Press the + button for create a virtual machine;
6. Write the name of machine, later press a 'Create' button;
7. Download an .iso image of any Windows, for example Windows 95;
8. Configure a VM, create .qcow2 disk, put a .iso in machine menu, and press > button for start a VM;
9.Enjoy :);
